# NutriSense App

A premium, interactive nutrition and calorie tracking application built with React and Vite.

## Features
- **Interactive Dashboard**: Real-time calorie tracking with dynamic macro sliders.
- **Meal Planning**: Detailed meal lists with nutrition facts and recipe links.
- **Mobile-First Design**: Optimized for a premium iOS mobile experience.
- **Action Feedback**: Visual notifications for all user interactions.

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Start the dev server with `npm run dev`.
